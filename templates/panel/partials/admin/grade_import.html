📖 Detailed Import Format Guide
{# CSV Format #}

CSV Format

The file must be a .csv with the following columns (header row required):

Column Description Example
student_id Student number (must exist in system) 2021-0001
subject_code Subject code (must exist and be enrolled) IT 211
grade Numeric grade or remark 1.5 or INC

Sample CSV

student_id,subject_code,grade
2021-0001,IT 211,1.5
2021-0002,IT 211,2.0
2021-0003,IT 212,INC
{# Excel Format #}

Excel Format (.xlsx)

The first sheet should have headers in the first row. Flexible column name matching:

  • Student ID: "Student ID", "student_id", "ID"
  • Subject Code: "Subject Code", "subject_code", "Code"
  • Grade: "Grade", "Final Grade"
{# PDF Format #}

PDF Format

The PDF should contain a table with grade data. The system will attempt to extract tables automatically. Best results with clear, well-structured tables.

{# Grade Values #}

Valid numeric grades: 1.0 · 1.25 · 1.5 · 1.75 · 2.0 · 2.25 · 2.5 · 2.75 · 3.0 · 5.0

Valid remarks: INC (Incomplete) · DRP (Dropped)

Note: The import is all-or-nothing — if any row has an error, no grades are saved.
